Workflows state machines that chain one-shot agents over board tasks

⛓ Workflows are state machines

Each step is a board-task spec. When its task finishes, the machine takes the on done or on fail transition — branch into remediation steps, loop back for visit-capped retries, and halt wherever you choose. Trigger runs by hand or on a cron.
